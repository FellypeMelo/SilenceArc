# Specification: Intel Arc GPU Optimization (SYCL)

**Track ID:** intel-arc-gpu-sycl_20260309
**Type:** Feature
**Created:** 2026-03-09
**Status:** Draft

## Summary

Migrate the DeepFilterNet noise suppression inference engine to the Intel Arc GPU using SYCL to reduce CPU overhead and leverage hardware acceleration (XMX).

## Context

Current inference is CPU-bound using the `tract` library in Rust. The build system already supports SYCL, and benchmarks confirm GPU execution is possible.

## User Story

As a user, I want the noise suppression to run on my Intel Arc GPU so that my CPU is free for other tasks and I get better performance.

## Acceptance Criteria

- [ ] Inference pipeline runs successfully on Intel Arc GPU.
- [ ] DeepFilterNet layers (GRU/Convolutions) utilize SYCL kernels.
- [ ] Measured reduction in CPU usage during active suppression.
- [ ] Memory management implemented using SYCL USM (Unified Shared Memory).

## Dependencies

- Requires modifications to the `DeepFilterNet/libDF` Rust codebase.
- Depends on the existing Intel OneAPI/SYCL integration in `CMakeLists.txt`.

## Out of Scope

- CPU Fallback logic (focus is exclusively on Intel GPU SYCL testing).
- Support for non-Intel GPUs.
